# Navegador Académico – Proyecto Final Estructura de Datos

## Descripción  
El **Navegador Académico** es una aplicación en lenguaje C que permite a los estudiantes universitarios registrar, visualizar y analizar su avance curricular de forma clara y automatizada.  
El sistema carga la malla curricular desde un archivo, guarda las asignaturas cursadas, calcula notas y promedios, valida prerrequisitos y muestra una visión global del progreso académico.

## Características principales  
- Carga de malla curricular desde archivo `.csv` o similar.  
- Registro de semestres y asignaturas cursadas.  
- Ingreso de notas con ponderaciones (componentes) y cálculo automático de la nota final de cada asignatura.  
- Cálculo de promedio ponderado por semestre y promedio general acumulado (PGA).  
- Validación de prerrequisitos antes de inscripción de asignaturas.  
- Listado de asignaturas aprobadas, pendientes y disponibles para inscribir.  
- Exportación de reporte académico en archivo externo.
